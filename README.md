# Course Approval System using Flask

## 📌 Project Description
This project is a web-based Course Approval System developed using Python Flask and HTML.  
It allows students to submit course requests and teachers to approve or reject them.

---

## ⚙️ Technologies Used
- Python  
- Flask (Web Framework)  
- HTML  
- Session Management (for login system)

---

## 🎯 Features
- Student login system  
- Course submission form  
- Teacher login system  
- Approve / Reject functionality  
- Status tracking (Pending / Approved / Rejected)

---

## 🔐 Login Details
- Teacher Password: `admin123`

---

## 🔄 How It Works
1. Student logs in  
2. Submits course request  
3. Request stored in system  
4. Teacher logs in  
5. Teacher approves or rejects request  
6. Status gets updated
